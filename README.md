# Urna Eletrônica – Java

Projeto acadêmico desenvolvido em Java com o objetivo de simular o funcionamento básico de uma urna eletrônica.

## Funcionalidades
- Exibição de candidatos para senador e presidente
- Registro de votos
- Validação de votos (válidos, nulos e em branco)
- Cálculo de totais e percentuais

## Tecnologias utilizadas
- Java
- Lógica de Programação
- Estruturas condicionais e de repetição

## Objetivo
Projeto criado durante a graduação em Engenharia de Software para praticar lógica de programação e organização de código, visando aprendizado contínuo e oportunidade de estágio em TI.
